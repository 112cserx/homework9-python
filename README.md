# homework9-python
# Анализ событий информационной безопасности

## Цель работы

Закрепить навыки работы с библиотеками Pandas, Matplotlib и Seaborn, а также научиться загружать, анализировать и визуализировать данные в Python.

---

## Описание данных

В проекте используется файл `events.json`, содержащий события информационной безопасности.

Каждое событие включает:
- timestamp — дата и время события
- signature — тип события (тип атаки)

---

## Этап 1. Загрузка данных

Данные из JSON-файла были загружены в Python и преобразованы в DataFrame с использованием библиотеки Pandas.

Использовался следующий подход:

1. Чтение JSON-файла через модуль json
2. Извлечение списка событий из ключа "events"
3. Преобразование списка в DataFrame

После загрузки данные содержат 100 записей и 2 столбца:
- timestamp
- signature

---

## Этап 2. Анализ данных

Для анализа распределения типов событий использован метод:

df["signature"].value_counts()

Этот метод позволяет определить:
- сколько раз встречается каждый тип события;
- какой тип события встречается чаще всего;
- общее распределение событий по категориям.

По результатам анализа выявлено, что один из типов атак встречается чаще остальных, а остальные распределены примерно равномерно.

---

## Этап 3. Визуализация данных

Для визуализации использована библиотека Seaborn.

Построен столбчатый график распределения типов событий:

- по оси X — тип события (signature)
- по оси Y — количество событий

График позволяет наглядно оценить структуру распределения атак.
